# umi4 pro web
一个基于最新的umi4搭建的中台模版项目

## 功能合集：
1. 基于umi4，是用了其中许多的配置
2. 使用react-activation实现了多标签页缓存
3. 使用mobx作为数据管理
4. 使用config作为配置，并且添加l自定义插件配置
5. 使用antd的Layout作为基础布局
6. 使用react-transition-group和react-nprogress作为路由切换动画

## 安装
使用本仓库地址克隆项目到本地，然后安装依赖
```bash
git clone 仓库地址
```
安装依赖
```bash
yarn install
```
启动项目
```bash
yarn start
```
